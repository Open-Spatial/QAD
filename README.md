# QAD_QT6

Open Spatial's Qt6 / QGIS 4 fork of the QAD Quantum Aided Design plugin.

The installable QGIS plugin package is in `QAD_QT6/`. See
`QAD_QT6/README.md` for installation, development, license, and attribution
details.

This fork is based on the original QAD project at https://github.com/gam17/QAD
and preserves the upstream GPL notices and author credits.
